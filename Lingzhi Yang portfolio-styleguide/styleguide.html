Styleguide

Visual Identity System

This styleguide defines the visual language of my portfolio through typography, color, hierarchy, image treatment, and reusable interface elements. It translates my creative identity into a consistent digital system.

Brand Statement

Rebuilding the world with pixels and inspiration.

Bold Editorial Interactive Cinematic

01

Brand Overview

Ethos

My portfolio combines visual storytelling, interaction, motion, and structured web design. The visual identity is expressive but controlled, using contrast, strong typography, and selective color to create a memorable experience.

Audience

The primary audience includes instructors, collaborators, and future employers in fields related to digital media, web design, motion design, and interactive arts.

Design Direction

The system is inspired by editorial composition, poster-like scale, cinematic imagery, and clean web structure. It is meant to feel personal, contemporary, and visually confident.

03

Typography

Display Typeface

Anton

Used for hero titles, major headings, marquee text, and statement-driven content. It creates impact, confidence, and poster-like presence.

Body Typeface

Inter is used for readable body text, navigation, descriptive content, and interface clarity.

The contrast between Anton and Inter supports both strong visual identity and functional readability.

Type Hierarchy

H1

Used for page titles and major statements.

H2

Used for section headings and structured page flow.

Body

Used for biography, descriptions, and process writing.

Label

Used for section tags, numbering, and small headings.

Layout Rules

Spacing

Large vertical spacing is used to separate sections and create a calm editorial rhythm.

Contrast

Black text on a soft neutral background creates clarity, while accent colors are used selectively.

Hierarchy

Pages rely on large titles, restrained labels, and clean supporting text to guide the viewer.

Responsiveness

The system adapts from wide editorial layouts on desktop to simplified stacked structures on mobile.
